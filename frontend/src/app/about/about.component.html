About this project

Once upon a time, I worked with backups and I frequently needed to create tape labels. One day, the free site I used started charging money for the service so I developed the predecessor to this project.

The second iteration of this project was written in JavaScript using AngularJS with a backend in Node.JS. Many years and countless iterations later I'm still using Node.JS on the backend, but everything is TypeScript, and the frontend is Angular.

I finally decided to take the time to publish this as a standalone project and publish it to GitHub. Enjoy.
